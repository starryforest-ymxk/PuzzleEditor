# 面包屑下划线悬停显示修复记录

## 背景
- 进入内部层级后，面包屑文字一直带下划线，违背“仅悬停时显示下划线”的视觉预期。

## 修改点
- 更新 `components/Layout/Breadcrumb.tsx`：默认移除下划线，仅在非末级节点鼠标悬停时动态添加下划线，末级保持普通文本；后退按钮改为基于浏览历史栈，始终可见（在根级亦可点击返回上一视图）。
- 样式层面补充 `.breadcrumb-link:hover` 规则（`styles.css`），避免点击后遗留下划线。

## 测试建议
- 进入 Stage/Node 层级，观察面包屑：
  - 默认无下划线。
  - 光标悬停非末级文字时出现下划线，移出后消失。
  - 末级文字始终无下划线。
